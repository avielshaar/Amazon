# Amazon Clone 🛒

A full-stack e-commerce application built with the MERN stack.

## Features
- Product listings with search and filtering
- Shopping cart with persistent state (Redux Toolkit)
- User authentication with JWT
- Responsive UI built with React-Bootstrap

## Tech Stack
**Frontend:** React, Redux Toolkit, React-Bootstrap, Axios  
**Backend:** Node.js, Express (MVC architecture)  
**Database:** MongoDB, Mongoose
